**ถาม: ผู้เล่นต้องลง mod ฝั่ง client ไหม?**
ตอบ: ไม่ต้อง ใช้ได้บน client ปกติ แค่ยอมรับ resource pack ของเซิร์ฟ

**ถาม: จำเป็นต้องมี packetevents จริงไหม?**
ตอบ: จำเป็นสำหรับเมนูเคอร์เซอร์ (ใช้ทำการแช่กล้อง) ถ้าไม่มีเมนูจะทำงานไม่ถูกต้อง

**ถาม: ต้องมี ItemsAdder ไหม?**
ตอบ: แนะนำอย่างยิ่ง ใช้ให้รูปปุ่ม/เคอร์เซอร์ และโมเดลรางวัล ถ้าไม่มีเมนูจะเป็นกล่อง (□)

**ถาม: ใช้กับ Spigot/Bukkit ได้ไหม?**
ตอบ: ไม่ได้ — ต้องเป็น **Paper** 1.21+ (ใช้ API เฉพาะ Paper)

**ถาม: เพิ่มตู้ใหม่ยังไง?**
ตอบ: คัดลอกบล็อกใน `crates.yml` ตั้ง id ใหม่ แก้ค่าเปิด/รางวัล แล้ว `/gacha reload`
ดู [ตู้กาชา](crates.html)

**ถาม: เปลี่ยนโอกาสดรอปยังไง?**
ตอบ: แก้ `weight` ของแต่ละเรตใน `config.yml` weight สูง = ออกบ่อย ดู [การตั้งค่า](configuration.html)

**ถาม: token คืออะไร?**
ตอบ: สกุลเงินในตัว (เก็บใน `tokens.yml`) ขายจากร้านได้ ให้ด้วย `/gacha token give <player> <amount>`
ตู้ตั้งค่าเปิดเป็น `TOKEN` ได้

**ถาม: ร้านค้าออนไลน์ให้ token อัตโนมัติได้ไหม?**
ตอบ: ได้ ตั้งให้ร้านรัน `gacha token give {player} <amount>` จาก console ตอนซื้อ

**ถาม: "Open Again" คิดเงินไหม?**
ตอบ: คิด เท่ากับการเปิดใหม่ตามจำนวนนั้น ไม่ข้ามค่าเปิด

**ถาม: ทำไม Back เป็นสีเทาตอนเปิด 10 ใบ?**
ตอบ: ตั้งใจให้เป็นแบบนั้น ต้องเผยครบ 10 ใบก่อน ใช้ **Open All** เพื่อเผยทันที

**ถาม: ผู้เล่นหลุดกลางการเปิด รางวัลหายไหม?**
ตอบ: ไม่หาย รางวัลถูกบันทึกทันทีที่สุ่มได้ และแจกตอน join/respawn ครั้งถัดไป

**ถาม: ปุ่ม/ข้อความอยู่ผิดที่ ย้ายยังไง?**
ตอบ: แก้ `theme.yml` (และ `THEME-REFERENCE.md` สำหรับ reference เต็ม) แล้ว `/gacha reload`

**ถาม: ข้อความเมนูในแชตเป็นภาษาไทยเพี้ยน เพราะอะไร?**
ตอบ: เป็นที่ font ของ resource pack ไม่รองรับสระไทยซ้อน ไม่ใช่บั๊กปลั๊กอิน
ใช้ข้อความอังกฤษ หรือเพิ่ม font ไทยใน pack

**ถาม: ขยับเคอร์เซอร์ให้เร็ว/ถึงขอบยังไง?**
ตอบ: เพิ่ม `sensitivity-x` และขยาย `bounds` ใน `cursor.yml` ดู [Cursor UI](cursor-ui.html)

**ถาม: reload หลังแก้ไฟล์ยังไง?**
ตอบ: `/gacha reload` สำหรับการแก้ YAML รีสตาร์ทเต็มต้องทำเฉพาะตอนเปลี่ยนไฟล์ jar
