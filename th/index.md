# CradGacha



---

## CradGacha คืออะไร?

**CradGacha** คือปลั๊กอินกาชา (กล่องสุ่มของรางวัล) สำหรับเซิร์ฟเวอร์ Minecraft **Paper 1.21+**

แทนที่จะเป็นหน้าต่างหีบแบบปกติ มันจะแสดง **เมนูโฮโลแกรมลอยในโลก** และให้ผู้เล่นมี
**เคอร์เซอร์เมาส์เสมือน** ที่ขยับด้วยเมาส์เพื่อชี้และคลิกได้ — เหมือน UI ของเกมจริง ๆ
ผู้เล่นเลือกตู้ เปิด **x1** หรือ **x10** พลิกการ์ดเพื่อดูรางวัล แล้ว **เปิดอีกครั้ง** ได้โดยไม่ต้องออกจากจอ

มันคือเวอร์ชันแยกที่โฟกัสเฉพาะ cursor gacha ออกจากโปรเจ็คเดิม สร้างใหม่ให้สะอาดและตรงประเด็น

## ใช้ทำอะไร?

- ทำระบบ **กล่องสุ่ม / กาชา** บนเซิร์ฟเวอร์
- ให้ผู้เล่นได้ UI สวยทันสมัยเหมือน custom client — แต่ใช้ได้บน **client ปกติ** (พร้อม resource pack)
  โดยไม่ต้องลง mod ฝั่ง client
- ขายการเปิดด้วย **เงิน (Vault)** หรือ **token** (สกุลเงินในตัว) ที่เติมจากร้านค้าได้

## เหมาะกับใคร?

- **เจ้าของเซิร์ฟ** ที่อยากได้ระบบกาชา/กล่องสุ่มสวย ๆ
- **แอดมินมือใหม่** — แก้แค่ไฟล์ YAML ไม่กี่ไฟล์ คู่มือนี้พาทำทีละขั้น
- **เซิร์ฟทุกแบบ** — มี ItemsAdder / Nexo / Oraxen ก็ภาพสวยสุด หรือไม่มีเลยก็ได้ (vanilla pack ในตัว render เมนูครบบน MC 1.21.4+)

## ฟีเจอร์

| ฟีเจอร์ | ทำอะไร |
|---|---|
| **เมนูกาชาแบบเคอร์เซอร์** | เมนูโฮโลแกรม + เคอร์เซอร์เมาส์เสมือน (ชี้แล้วคลิกในโลก) |
| **UI โฮโลแกรม/Display** | สร้างจาก Display entity ของ Minecraft (ข้อความ/ไอเทม/รูป) — ไม่มีหน้าต่างหีบ |
| **เลือกตู้** | รายการตู้แนวตั้งทางซ้าย คลิกเพื่อเลือก |
| **เปิด x1 / x10** | เปิดทีละใบ หรือสิบใบรวด จำนวนปรับได้ (`max-open`) |
| **การ์ดผลรางวัล** | การ์ดพลิกด้วยอนิเมชันแกน Y เพื่อเผยของที่ได้ |
| **Open All (x10 เท่านั้น)** | เปิด 10 ใบ → เผยการ์ดที่เหลือทั้งหมดในคลิกเดียว |
| **Open Again** | สุ่มใหม่ด้วย **จำนวนเดิม** (ยังคิดค่าเปิดตามปกติ) |
| **ปุ่ม Back** | กลับเมนูเลือกตู้ — แต่ตอนเปิด 10 ใบจะกดได้**หลังเปิดครบทั้ง 10** |
| **เศรษฐกิจ/ค่าเปิด** | ตู้แต่ละใบมีค่าเปิดเป็น `NONE` / `ITEM` / `MONEY` (Vault) / `TOKEN` (แต้มในตัว) |
| **ระบบรางวัล** | รางวัลให้ไอเทม รันคำสั่ง หรือทั้งคู่ พร้อมน้ำหนักเรตและระบบ pity |
| **Tokens** | สกุลเงินเติมเงินในตัว เก็บใน `tokens.yml` จัดการด้วย `/gacha token …` |
| **ระบบ Theme** | `theme.yml` ให้ย้าย/แต่งปุ่มและข้อความได้โดยไม่ต้องแตะโค้ด |
| **เช็คระบบเอง** | `/gacha doctor` ตรวจการตั้งค่า (packetevents, ItemsAdder, glyph, Vault, ตู้, resource pack) |
| **ความปลอดภัยรางวัล** | รางวัลที่สุ่มแล้วถูกบันทึกลงดิสก์ทันที จึงไม่หายแม้เซิร์ฟดับ/หลุด/ตาย |

> ⭐ **พรีเมียม** เพิ่ม editor & สร้างตู้ในเกม, rate-up event, VIP luck, กุญแจตู้, หน้าสถิติ, layout editor
> และ Developer API (และปลดล็อกจำกัด 3 ตู้) ดูหน้า [พรีเมียม](/th/premium)

## เริ่มเร็ว (ฉบับย่อ)

1. ติดตั้ง **packetevents** (จำเป็น) และ **Vault** ถ้าจะขายด้วยเงิน · item provider (ItemsAdder / Nexo / Oraxen) ไม่บังคับ
2. วาง **CradGacha jar** ลงโฟลเดอร์ `plugins/` แล้วเปิดเซิร์ฟ
3. รัน **`/gacha setup`** — ติดตั้ง resource pack ให้ตรง provider (หรือ vanilla ถ้าไม่มี)
4. รัน `/gacha doctor` เช็คว่าทุกอย่างเขียว แล้ว `/gacha` เปิดเมนู

ปลั๊กอินฝัง resource pack มาให้และแตกออกครั้งแรกที่เปิด — แค่รัน [`/gacha setup`](setup.md)
รายละเอียดเต็มอยู่ที่หน้า [การติดตั้ง](installation.md) และ [ตั้งค่าครั้งแรก](first-setup.md)
